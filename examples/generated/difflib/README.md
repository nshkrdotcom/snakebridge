# SnakeBridge.Difflib Examples

Auto-generated examples for the `difflib` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Difflib.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 8 available functions
- `stateless_examples.exs` - Examples of 3 pure functions
- `stateful_examples.exs` - Examples of 5 stateful functions

## Running Examples

```bash
mix run examples/generated/difflib/all_functions.exs
mix run examples/generated/difflib/stateless_examples.exs
```

## Stateless Functions (3)

Pure functions with no side effects:

- `IS_CHARACTER_JUNK`
- `IS_LINE_JUNK`
- `ndiff`


## Stateful Functions (5)

Functions that may have side effects:

- `context_diff`
- `diff_bytes`
- `get_close_matches`
- `restore`
- `unified_diff`
