# SnakeBridge.Requests Examples

Auto-generated examples for the `requests` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Requests.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 10 available functions
- `stateless_examples.exs` - Examples of 1 pure functions
- `stateful_examples.exs` - Examples of 9 stateful functions

## Running Examples

```bash
mix run examples/generated/requests/all_functions.exs
mix run examples/generated/requests/stateless_examples.exs
```
