# SnakeBridge.Statistics Examples

Auto-generated examples for the `statistics` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Statistics.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 19 available functions
- `stateless_examples.exs` - Examples of 17 pure functions
- `stateful_examples.exs` - Examples of 2 stateful functions

## Running Examples

```bash
mix run examples/generated/statistics/all_functions.exs
mix run examples/generated/statistics/stateless_examples.exs
```
