# SnakeBridge.Textwrap Examples

Auto-generated examples for the `textwrap` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Textwrap.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 5 available functions
- `stateless_examples.exs` - Examples of 4 pure functions
- `stateful_examples.exs` - Examples of 1 stateful functions

## Running Examples

```bash
mix run examples/generated/textwrap/all_functions.exs
mix run examples/generated/textwrap/stateless_examples.exs
```

## Stateless Functions (4)

Pure functions with no side effects:

- `fill`
- `indent`
- `shorten`
- `wrap`


## Stateful Functions (1)

Functions that may have side effects:

- `dedent`
