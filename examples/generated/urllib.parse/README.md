# SnakeBridge.UrllibParse Examples

Auto-generated examples for the `urllib.parse` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.UrllibParse.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 28 available functions
- `stateless_examples.exs` - Examples of 17 pure functions
- `stateful_examples.exs` - Examples of 11 stateful functions

## Running Examples

```bash
mix run examples/generated/urllib.parse/all_functions.exs
mix run examples/generated/urllib.parse/stateless_examples.exs
```
