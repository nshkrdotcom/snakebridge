# SnakeBridge.Uuid Examples

Auto-generated examples for the `uuid` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Uuid.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 6 available functions
- `stateless_examples.exs` - Examples of 4 pure functions
- `stateful_examples.exs` - Examples of 2 stateful functions

## Running Examples

```bash
mix run examples/generated/uuid/all_functions.exs
mix run examples/generated/uuid/stateless_examples.exs
```

## Stateless Functions (4)

Pure functions with no side effects:

- `getnode`
- `uuid3`
- `uuid4`
- `uuid5`


## Stateful Functions (2)

Functions that may have side effects:

- `main`
- `uuid1`
