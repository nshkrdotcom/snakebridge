# SnakeBridge.Base64 Examples

Auto-generated examples for the `base64` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Base64.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 21 available functions
- `stateless_examples.exs` - Examples of 14 pure functions
- `stateful_examples.exs` - Examples of 7 stateful functions

## Running Examples

```bash
mix run examples/generated/base64/all_functions.exs
mix run examples/generated/base64/stateless_examples.exs
```
