# SnakeBridge.Hashlib Examples

Auto-generated examples for the `hashlib` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Hashlib.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 2 available functions
- `stateless_examples.exs` - Examples of 1 pure functions
- `stateful_examples.exs` - Examples of 1 stateful functions

## Running Examples

```bash
mix run examples/generated/hashlib/all_functions.exs
mix run examples/generated/hashlib/stateless_examples.exs
```
