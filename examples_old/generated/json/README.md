# SnakeBridge.Json Examples

Auto-generated examples for the `json` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Json.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 5 available functions
- `stateless_examples.exs` - Examples of 3 pure functions
- `stateful_examples.exs` - Examples of 2 stateful functions

## Running Examples

```bash
mix run examples/generated/json/all_functions.exs
mix run examples/generated/json/stateless_examples.exs
```

## Stateless Functions (3)

Pure functions with no side effects:

- `detect_encoding`
- `dumps`
- `loads`


## Stateful Functions (2)

Functions that may have side effects:

- `dump`
- `load`
