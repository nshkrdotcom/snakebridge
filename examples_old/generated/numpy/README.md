# SnakeBridge.Numpy Examples

Auto-generated examples for the `numpy` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Numpy.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 50 available functions
- `stateless_examples.exs` - Examples of 21 pure functions
- `stateful_examples.exs` - Examples of 29 stateful functions

## Running Examples

```bash
mix run examples/generated/numpy/all_functions.exs
mix run examples/generated/numpy/stateless_examples.exs
```
