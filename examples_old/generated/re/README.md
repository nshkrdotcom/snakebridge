# SnakeBridge.Re Examples

Auto-generated examples for the `re` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Re.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 12 available functions
- `stateless_examples.exs` - Examples of 10 pure functions
- `stateful_examples.exs` - Examples of 2 stateful functions

## Running Examples

```bash
mix run examples/generated/re/all_functions.exs
mix run examples/generated/re/stateless_examples.exs
```

## Stateless Functions (10)

Pure functions with no side effects:

- `escape`
- `findall`
- `finditer`
- `fullmatch`
- `match`
- `purge`
- `search`
- `split`
- `sub`
- `subn`


## Stateful Functions (2)

Functions that may have side effects:

- `compile`
- `template`
