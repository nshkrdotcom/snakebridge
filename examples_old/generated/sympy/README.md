# SnakeBridge.Sympy Examples

Auto-generated examples for the `sympy` Python library.

## Quick Start

```elixir
# In IEx
iex> SnakeBridge.Sympy.function_name(args)
```

## Files

- `all_functions.exs` - Lists all 50 available functions
- `stateless_examples.exs` - Examples of 24 pure functions
- `stateful_examples.exs` - Examples of 26 stateful functions

## Running Examples

```bash
mix run examples/generated/sympy/all_functions.exs
mix run examples/generated/sympy/stateless_examples.exs
```
